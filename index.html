My Projects

Amazon Clone

The Amazon Clone project aims to replicate key features of the Amazon e-commerce platform. This involves developing a web application where users can browse products, search for items, add them to their shopping cart, and complete a purchase. The project will also include features such as user authentication, product reviews, order tracking, and an admin panel for managing the platform.

Tic Tac Toe Game

The Tic Tac Toe project is a classic two-player game where the objective is to get three of your marks (X or O) in a row, either horizontally, vertically, or diagonally. The game is simple yet engaging and serves as an excellent introduction to game development concepts such as game logic, user interface design, and event handling.

To - Do List

The To - Do List project is a web application that allows users to manage their daily tasks efficiently. The primary goal is to provide users with a simple interface to create, edit, and organize their to-do items. Users can mark tasks as completed, prioritize them, and categorize them for better management.
